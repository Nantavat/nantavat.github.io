คำถามคำตอบ
1.css sytax ประกอบด้วยอะไรบ้างselector, property และ value
2.การเขียน CSS มีกี่แบบอะไรบ้างคำตอบ มี3แบบ css inline, internal, external
3.Selectors มีกี่แบบอะไรบ้างคำตอบ มี3แบบ Class Selectors,ID Selectors,TAG Selectors
4.combinator มีกี่แบบอะไรบ้างคำตอบ 1. Simple selectors 2.Pseudo-elements 3.Combinators
5.property และ value คืออะไรแตกต่างกันอย่างไรvalue คือค้่าของ property
6.property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง1.Content eg. font-size,font-weigh,font-family,color 2.Border eg. border table 3.Padding eg. padding-left 4.Margin margin-left margin-right margin-top margin-bottom 5.Element eg. height,weight,flex,grid
7.value unit มีกี่ประเภทอะไรบ้างแต่ละ ประเภทคืออะไร1. Absolute eg px 2. Relative eg rem em
